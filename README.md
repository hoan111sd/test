# ComesticsShopApi
    
#Folder Structure

```
├── app                         //Nodejs source files
│   ├── backgroundJob  
│   ├── config                  //Config passportJs for login and register
│   ├── controlers              //Apis
│   ├── functions               //Global functions
│   ├── middleware              //Middleware authentication
│   ├── public                  //CSS, image ...
│   └── views                   //Views admin page
├── config                      //Config
│   └── index.js                
├── config.js
├── e2e
├── package.json
├── server.js
├── src
└── yarn.lock
```

#Installation

Install nodejs, yarn, git, mongoDb.
```
    git clone https://github.com/burdy035/ComesticsShopApi.git
    yarn global add @angular/cli or npm install -g @angular/cli
    yarn install or npm install
    ng build
    node server.js
```
